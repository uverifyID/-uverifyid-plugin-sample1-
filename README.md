# react-native-uverifyid

react native plugin for UverifyID SDK

## Installation

```sh
npm install react-native-uverifyid
```

## Usage

```js
import Uverifyid from "react-native-uverifyid";

// ...

const result = await Uverifyid.multiply(3, 7);
```
